## VUE3.0初学  

#### vue2.x与vue3.0的差别

- 响应式原理

  - **vue2.x**   通过Object.defineProperty劫持数据的getter和setter

    - **缺点：**  只能拦截对象中初始的属性，检测不到对象**增加或删除**的属性，不过可以通过$set和$delete方法实现；如果要监听一个深度嵌套对象，需要使用watch+deep属性，而且会将这个嵌套对象的每一层都变成响应式，若是嵌套对象过于复杂就会产生性能问题
- **vue3.0** 通过Proxy API 做数据劫持。
    - **优点：** 可以监听对象中属性的增加与删除
    - **缺点：** Proxy API 无法监听内部深层次的对象变化，解决方案是在getter中去递归响应式，这样只有被访问使用的特定深层次对象才会变成响应式，相比Object.defineproperty提升了性能
